# TaskBase

**TaskBase** is a task management application built with the modern web stack. It features secure authentication, real-time data synchronization with optimistic UI updates, and a polished, responsive user interface.

-   **Authentication**: Secure Sign Up and Log In powered by Supabase Auth.
